# MILO.ONE

MILO.ONE is an experimental research platform for structured human–AI interaction.  
It operates as the technical environment of the preregistered research program  
**Computational Behavioral Linguistics (CBL)**.

📄 Preregistration (Zenodo):  
https://doi.org/10.5281/zenodo.18878139

---

## Core Dimensions

- **Control**  
- **Regulation**  
- **Embodiment**

These form the theoretical backbone of the CoRE framework, which investigates  
behavioral structures in human–AI interaction.

---

## Current Work

- CoRE framework architecture  
- psycholinguistic text-analysis pipelines (~180 linguistic feature categories)  
- longitudinal drift measurement for large language models  
- behavioral stylometry (PCA, SEM, K-Factor)  
- experimental dialogic AI models (MILO.ONE instances)
