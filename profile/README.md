# MILO.ONE

MILO.ONE is an experimental research project exploring structured human–AI interaction.

The project focuses on three core dimensions:

- Control  
- Regulation  
- Embodiment  

Current work includes:

- CoRE framework architecture
- psycholinguistic analysis pipelines
- experimental dialogic AI models
